# Initial Concept
A mini social media application inspired by GitHub, built for learning frontend and backend development.

# Product Definition

## Vision
A social platform that mimics GitHub's social interaction patterns—posts, comments, follows, and "stars"—to provide a familiar yet distinct social media experience. The goal is to create a robust, high-performance web application that serves as a learning project for full-stack development.

## Target Audience
- Developers and tech enthusiasts looking for a GitHub-like social experience.
- Users interested in a minimalist, text-and-code-centric social platform.
- The creator (for learning and portfolio purposes).

## Core Goals
- Implement a responsive and intuitive user interface using modern React patterns.
- Ensure secure and flexible authentication using Better Auth.
- Optimize data fetching and state management with TanStack Query and Zustand.
- Build a scalable backend with Next.js Server Actions and Prisma/MongoDB.

## Key Features
- **User Profiles:** Personalized profiles showing user activity, posts, and followers.
- **Social Feed:** A dynamic feed of posts from followed users and recommendations.
- **Post Creation:** Rich text or Markdown-based post editor.
- **Interactions:** "Starring" (liking) posts, commenting, and following users.
- **Search:** Discovering users and content through an efficient search mechanism.
