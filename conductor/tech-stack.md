# Technology Stack

## Core Technologies
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Frontend Framework:** [Next.js 16](https://nextjs.org/) (App Router) with [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/), [Shadcn UI](https://ui.shadcn.com/), [Radix UI](https://www.radix-ui.com/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/), [TanStack Query](https://tanstack.com/query/latest)
- **Backend & ORM:** [Next.js Server Actions](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations), [Prisma ORM](https://www.prisma.io/)
- **Database:** [MongoDB](https://www.mongodb.com/)
- **Authentication:** [Better Auth](https://www.better-auth.com/)

## Development Tools
- **Package Manager:** npm
- **Linting:** ESLint
- **Formatting:** Prettier
- **Database Management:** Prisma Studio
